# Определение перспективного тарифа для телеком-компании

**Цель проекта:** используя данные о пользователях оператора сотовой связи, проанализировать поведение клиентов и дать рекомендации по выбору оптимального тарифа.

**Выводы:**

  - Клиенты премиального тарифа имеют более разнообразный профиль расходов на мобильную связь, например, могут практически не пользоваться пакетом услуг, при этом продолжая платить высокую абонентскую плату. В целом, по основным показателям их средняя активность выше.

  - Пользователи стандартного тарифа часто не укладываются в базовый пакет услуг. Однако, мало кому из выборки переход на премиальный тариф позволил бы сэкономить деньги.

  - Объем пакетов минут и сообщений в премиальном тарифном плане избыточен - большинство пользователей не используют и половины.

  - В результате проверки гипотезы о разнице средней выручки премиального и стандартного тарифов показано, что наибольший доход компании приносят абоненты премиального тарифного плана, особенно учитывая то, что они редко пользуются пакетом услуг полноценно.

  - В результате проверки гипотезы о разнице выручки от пользователей из Москвы и абонентов из других регионов обнаружено, что разница доходов от пользователей из Москвы и из регионов статистически не значима.

**Стек технологий:** python, pandas, matplotlib, seaborn, scipy
